# Тесты для практических заданий

Эта папка содержит тесты для проверки решений практических заданий из инструкции.

## Как запускать тесты

```bash
# Из корневой папки проекта
python tests/test_level1.py
python tests/test_level2.py
python tests/test_level3.py
```

## Структура тестов

- `test_level1.py` - тесты для заданий уровня 1 (легко)
- `test_level2.py` - тесты для заданий уровня 2 (средне)
- `test_level3.py` - тесты для заданий уровня 3 (сложно)
- `test_helpers.py` - вспомогательные функции для тестов

## Примечание

Некоторые тесты проверяют только структуру кода, а не визуальный результат.
Для полной проверки нужно запустить игру и проверить изменения визуально.
