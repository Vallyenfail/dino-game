# Как запускать тесты

Эта папка содержит тесты для проверки выполнения практических заданий.

## Быстрый старт

### Запуск всех тестов сразу

**Windows:**
```bash
cd C:\путь\к\dino-game
python tests\test_level1.py
python tests\test_level2.py
python tests\test_level3.py
```

**macOS/Linux:**
```bash
cd ~/путь/к/dino-game
python3 tests/test_level1.py
python3 tests/test_level2.py
python3 tests/test_level3.py
```

### Запуск конкретного уровня

**Уровень 1 (Легко):**
```bash
python tests/test_level1.py
```

**Уровень 2 (Средне):**
```bash
python tests/test_level2.py
```

**Уровень 3 (Сложно):**
```bash
python tests/test_level3.py
```

## Что проверяют тесты?

### Уровень 1 - Персонализация
- ✅ Наличие файлов динозавра
- ✅ Наличие музыкальных файлов
- ✅ Определение цветов кнопок
- ✅ Настройки размера окна
- ✅ Функция создания кактусов
- ✅ Наличие звука прыжка

### Уровень 2 - Изменение механики
- ✅ Параметр здоровья
- ✅ Скорость пуль
- ✅ Массив кактусов
- ✅ Логика сердечек
- ✅ Скорость птиц
- ✅ Функция проверки урона от птиц
- ✅ Функция прыжка

### Уровень 3 - Новая функциональность
- ✅ Базовая структура для урона от птиц
- ✅ Система сохранений
- ✅ Обработка клавиши ESC
- ✅ Функция выбора героя
- ✅ Базовая структура для нового врага
- ✅ Система подсчета очков
- ✅ Логика бонусов
- ✅ Система сохранения максимального счета
- ✅ Функция паузы
- ✅ Функции проверки здоровья и game over

## Примечание

⚠️ **Важно:** Тесты проверяют только наличие базовой структуры кода, а не визуальный результат.

Для полной проверки заданий нужно:
1. Запустить игру
2. Проверить изменения визуально
3. Протестировать новую функциональность вручную

## Интерпретация результатов

- ✅ **PASSED** - тест пройден успешно
- ❌ **FAILED** - тест не пройден, нужно проверить код

Если тест не прошел, это может означать:
- Файл не существует
- Функция не определена
- Параметр не найден в коде

## Решения для сложных заданий

Если задание кажется слишком сложным, посмотри готовые решения в папке `tests/solutions/`:

- `solution_3_2_save_system.py` - Система полноценных сохранений
- `solution_3_5_new_enemy.py` - Новый тип врага (летучая мышь)
- `solution_3_8_leaderboard.py` - Таблица лидеров

## Помощь

Если тесты не запускаются:

1. Убедись, что ты находишься в корневой папке проекта
2. Проверь, что Python установлен: `python --version`
3. Проверь, что pygame установлен: `pip list | grep pygame`
4. Убедись, что файлы проекта на месте

Если нужна помощь, спроси преподавателя!
